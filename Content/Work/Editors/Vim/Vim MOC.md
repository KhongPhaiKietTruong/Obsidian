# Vim MOC (Map of Content)

Chào mừng bạn đến với tài liệu hướng dẫn học và tra cứu Vim. Dưới đây là mục lục liên kết đến các phần nội dung chi tiết:

## 1. Chế độ Normal (Normal Mode)
- [[Content/Work/Editors/Vim/Normal mode/move|move]] - Di chuyển con trỏ cơ bản (`h`, `j`, `k`, `l` và di chuyển nhanh theo số dòng)
- [[Content/Work/Editors/Vim/Normal mode/word-based movement|word-based movement]] - Di chuyển con trỏ theo từ (`w`, `b`, `W`, `B`)
- [[Content/Work/Editors/Vim/Normal mode/go to the first line of the file|go to the first line of the file]] - Di chuyển về dòng đầu tiên của file (`gg`)
- [[Content/Work/Editors/Vim/Normal mode/go to the last line of the file|go to the last line of the file]] - Di chuyển về dòng cuối cùng của file (`G`)
- [[Content/Work/Editors/Vim/Normal mode/jump to the begining of line|jump to the begining of line]] - Nhảy về đầu dòng (`_`)
- [[Content/Work/Editors/Vim/Normal mode/jump to the end of line|jump to the end of line]] - Nhảy về cuối dòng (`$`)
- [[Content/Work/Editors/Vim/Normal mode/jump to the specified charater|jump to the specified charater]] - Nhảy nhanh đến ký tự chỉ định (`f`, `F`)
- [[Content/Work/Editors/Vim/Normal mode/jump down half-page|jump down half-page]] - Cuộn màn hình xuống nửa trang (`Ctrl + D`)
- [[Content/Work/Editors/Vim/Normal mode/jump up half-page|jump up half-page]] - Cuộn màn hình lên nửa trang (`Ctrl + I`)
- [[Content/Work/Editors/Vim/Normal mode/center the current line on the screen|center the current line on the screen]] - Đưa dòng hiện tại vào giữa màn hình (`zz`)
- [[Content/Work/Editors/Vim/Normal mode/change a word when the cursor is between the word|change a word when the cursor is between the word]] - Thay đổi từ chứa con trỏ (`ciw`, `ciW`)
- [[Content/Work/Editors/Vim/Normal mode/delete word|delete word]] - Xóa từ (`dw`, `dW`, `db`)
- [[Content/Work/Editors/Vim/Normal mode/delete line|delete line]] - Xóa dòng hiện tại (`dd`)
- [[Content/Work/Editors/Vim/Normal mode/delete multiple lines|delete multiple lines]] - Xóa nhiều dòng (`d + number + j/k`)
- [[Content/Work/Editors/Vim/Normal mode/delete from a cursor to specified character|delete from a cursor to specified character]] - Xóa từ con trỏ đến trước ký tự chỉ định (`d t <char>`)
- [[Content/Work/Editors/Vim/Normal mode/delete from the cursor to the end of line|delete from the cursor to the end of line]] - Xóa từ con trỏ đến cuối dòng (`d$`)
- [[Content/Work/Editors/Vim/Normal mode/delete all|delete all]] - Xóa toàn bộ nội dung file (`ggdG`)
- [[Content/Work/Editors/Vim/Normal mode/undo|undo]] - Hoàn tác các thay đổi (`u`, `U`)
- [[Content/Work/Editors/Vim/Normal mode/jump to the beginning of line and switch to insert mode|jump to the beginning of line and switch to insert mode]] - Nhảy về đầu dòng và chuyển sang chế độ chèn (`I`)
- [[Content/Work/Editors/Vim/Normal mode/jump to the end of line and switch to insert mode|jump to the end of line and switch to insert mode]] - Nhảy về cuối dòng và chuyển sang chế độ chèn (`A`)
- [[Content/Work/Editors/Vim/Normal mode/new line and insert mode|new line and insert mode]] - Tạo dòng mới và chuyển sang chế độ chèn (`o`, `O`)
- [[Content/Work/Editors/Vim/Normal mode/search forward|search forward]] - Tìm kiếm từ khóa về phía trước (`/`)
- [[Content/Work/Editors/Vim/Normal mode/search backward|search backward]] - Tìm kiếm từ khóa ngược về phía trước (`?`)
- [[Content/Work/Editors/Vim/Normal mode/search the specified word forward|search the specified word forward]] - Tìm nhanh từ dưới con trỏ về phía trước (`*`)
- [[Content/Work/Editors/Vim/Normal mode/search the specified word backward|search the specified word backward]] - Tìm nhanh từ dưới con trỏ ngược về phía trước (`#`)

## 2. Chế độ Insert (Insert Mode)
- [[Content/Work/Editors/Vim/Insert mode/increase indent one tab|increase indent one tab]] - Thêm thụt lề một tab (`Ctrl + T`)
- [[Content/Work/Editors/Vim/Insert mode/decrease indent one tab|decrease indent one tab]] - Giảm thụt lề một tab (`Ctrl + D`)
- [[Content/Work/Editors/Vim/Insert mode/delete a character|delete a character]] - Xóa ký tự trước con trỏ (`Ctrl + H`)
- [[Content/Work/Editors/Vim/Insert mode/delete the word before cursor|delete the word before cursor]] - Xóa từ trước con trỏ (`Ctrl + W`)
- [[Content/Work/Editors/Vim/Insert mode/delete from cursor to the begining|delete from cursor to the begining]] - Xóa từ con trỏ về đầu dòng (`Ctrl + U`)
- [[Content/Work/Editors/Vim/Insert mode/delete the current line|delete the current line]] - Xóa toàn bộ dòng hiện tại (`Ctrl + U`)
- [[Content/Work/Editors/Vim/Insert mode/run shortcut of normal mode while insert mode|run shortcut of normal mode while insert mode]] - Thực thi phím tắt chế độ Normal khi đang ở chế độ Insert

## 3. Chế độ Command (Command Mode)
- [[Content/Work/Editors/Vim/Command mode/insert before the cursor|insert before the cursor]] - Chuyển sang chế độ Insert trước con trỏ (`i`)
- [[Content/Work/Editors/Vim/Command mode/insert after the cursor|insert after the cursor]] - Chuyển sang chế độ Insert sau con trỏ (`a`)
- [[Content/Work/Editors/Vim/Command mode/save file|save file]] - Lưu file (`:w`)
- [[Content/Work/Editors/Vim/Command mode/quit file|quit file]] - Thoát file (`:q`)
- [[Content/Work/Editors/Vim/Command mode/save and quit|save and quit]] - Lưu file và thoát (`:wq`)
- [[Content/Work/Editors/Vim/Command mode/delete all|delete all]] - Xóa toàn bộ nội dung file (`%d`)

- [[Content/Work/Editors/Vim/Visual mode/delete|delete]] - Xóa vùng bôi đen (`d`)
