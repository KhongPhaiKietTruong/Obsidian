# HTML MOC (Map of Content)

Chào mừng bạn đến với tài liệu hướng dẫn học và tra cứu HTML. Dưới đây là mục lục liên kết đến các phần nội dung chi tiết:

## 1. Cơ bản & Cách sử dụng (Basics & How-To)
- [[cách chạy file html|Cách chạy file HTML]] - Cách khởi chạy hoặc mở file HTML trên trình duyệt (`npx live-server`)
- [[Content/Work/Languages/Frontend/HTML/Theory/comment|Comment]] - Cách ghi chú trong mã nguồn HTML (`<!-- comment -->`)

## 2. Thẻ HTML thông thường (Normal Tags)
- [[html|html]] - Thẻ root bao quát toàn bộ nội dung của trang web
- [[Content/Work/Languages/Frontend/HTML/Theory/Normal tag/head|head]] - Chứa các siêu dữ liệu (metadata) của trang web như `title`, `meta`, `link`
- [[title|title]] - Đặt tiêu đề cho trang web hiển thị trên thanh tab trình duyệt
- [[meta|meta]] - Khai báo thông tin phụ như bảng mã ký tự, cấu hình viewport cho SEO và hiển thị
- [[link|link]] - Kết nối các tài nguyên bên ngoài (như CSS, favicon) vào file HTML
- [[script|script]] - Kết nối hoặc nhúng mã Javascript vào trang HTML
- [[Inline & Block tag|Inline & Block tag]] - Phân biệt thẻ dạng Block (chiếm cả dòng) và thẻ dạng Inline (chỉ chiếm đủ không gian nội dung)
- [[div|div]] - Khối phân chia (division) dùng làm container bao bọc các phần tử khác
- [[p|p]] - Định nghĩa một đoạn văn bản (paragraph)
- [[h|h]] - Tạo các tiêu đề từ cấp độ `<h1>` đến `<h6>`
- [[a|a]] - Tạo siêu liên kết (hyperlink) tới trang web hoặc phần khác
- [[img|img]] - Hiển thị hình ảnh trên trang web với các thuộc tính `src`, `alt`
- [[button|button]] - Tạo nút bấm có thể tương tác
- [[input|input]] - Tạo ô nhập liệu cho người dùng (văn bản, số, mật khẩu, checkbox, radio, v.v.)
- [[iframe|iframe]] - Nhúng một trang web khác (bản đồ, video) vào trang hiện tại
- [[br|br]] - Xuống dòng (ngắt dòng) không cần thẻ đóng

## 3. Thuộc tính (Attributes)
- [[class|class]] - Thuộc tính đặt tên lớp để định dạng CSS hoặc thao tác JS, một thẻ có thể có nhiều class
- [[id|id]] - Mã định danh duy nhất cho phần tử, thường dùng để định dạng CSS riêng biệt hoặc truy xuất bằng Javascript
- [[style|style]] - Nhúng trực tiếp thuộc tính CSS (inline CSS) vào thẻ HTML
- [[lang|lang]] - Khai báo ngôn ngữ của trang web (hỗ trợ dịch tự động và tối ưu SEO)

## 4. Thẻ ngữ nghĩa (Semantic Tags)
- [[header|header]] - Định nghĩa phần đầu của trang web hoặc một vùng nội dung chuyên biệt (tốt cho SEO)
- [[nav|nav]] - Chứa các liên kết điều hướng, thường dùng làm menu trang web
- [[section|section]] - Chia trang web thành các phần/chủ đề chuyên biệt
- [[aside|aside]] - Định nghĩa vùng nội dung bên lề (sidebar)
- [[footer|footer]] - Định nghĩa phần chân trang (chứa thông tin bản quyền, liên hệ)
- [[Favicon|Favicon]] - Cài đặt biểu tượng đại diện của website hiển thị trên trình duyệt
