Là hàm dùng để tính sự chênh lệch giữa kết quả mà model dự đoán với kết quả đúng
mục tiêu của cost function là điều chỉnh trọng số **w** và bias **b** sao cho cost function đạt giá trị nhỏ nhất có thể
một cost function phổ biến:

$$
J(W,b)=\frac{1}{m}∑m​( \hat{y}^{(i)} − y^{(i)})^2
$$


vector $\hat{y}$ chứa các giá trị mà mà mô hình dự đoán ra, còn y là kết quả đúng
